Eco Car Pool System

Overview

The Eco Car Pool System is a web-based application designed to promote sustainable transportation through efficient carpooling. The system enables users to share rides, thereby reducing traffic congestion, fuel consumption, and environmental pollution. It aims to encourage eco-friendly commuting practices while providing a convenient and cost-effective travel solution.

This project has been developed with equal contribution by Anilrajbm and ashikams.


---

Objectives

Encourage shared transportation to reduce carbon emissions

Minimize traffic congestion and fuel usage

Provide a simple and efficient carpool management system

Promote sustainable and responsible commuting habits



---

Features

User-friendly interface for carpool operations

Ability to offer and search for carpool rides

Clear display of ride and route details

Scalable architecture for future enhancements

Modular design for easy maintenance and upgrades

Installation and Setup

Step 1: Clone the Repository

git clone https://github.com/Anilrajbm/Eco_Car_pool_system.git
cd Eco_Car_pool_system

Step 2: Install Dependencies

npm install

Step 3: Run the Application

npm run dev

The application will run locally on the port displayed in the terminal.
