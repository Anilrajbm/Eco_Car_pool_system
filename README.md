# EcoRide Bangalore - Frontend

React + Tailwind CSS frontend for the EcoRide system.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

## Run

Start the development server:
```bash
npm run dev
```
App runs on `http://localhost:5173`.

## Build

Build for production:
```bash
npm run build
```

## Features

-   **Home**: Landing page with project overview.
-   **Find Ride**: Search for carpools between hotspots.
-   **Map & Routes**: Visualize routes and eco-scores (Google Maps integration pending).
-   **Dashboard**: Real-time charts for AQI and Traffic.
-   **Admin**: Government dashboard for violations.
